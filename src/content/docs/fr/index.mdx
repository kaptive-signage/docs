## Qu'est-ce que Kaptive ?

Kaptive est une solution d'affichage dynamique interactive qui vous permet de créer et de publier votre contenu sur n'importe quel écran en quelques secondes.

### Configuration rapide

1. **Créer un compte** - Inscrivez-vous sur [app.kaptive.ch](https://app.kaptive.ch)
2. **Connecter un player** - Ajoutez un player Kaptive ou un web player à votre compte
3. **Créer du contenu** - Utilisez notre éditeur intuitif pour créer vos écrans
4. **Publier** - Diffusez instantanément votre contenu sur vos écrans
